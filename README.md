## Installation

```bash
npm i svg-to-inline
```

## Usage

```html
<script type="module">
  import "svg-to-inline/svg-to-inline.js";
</script>

<svg-to-inline
  path="images/logos/logo.svg"
  lazy="true"
  class-Name="my-class other-class"
  loading-Label="Loading"
></svg-to-inline>
```
