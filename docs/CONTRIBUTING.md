### Pre-requirements

- [git](https://git-scm.com)
- [nvm](https://github.com/nvm-sh/nvm)

### Install node

```bash
nvm install
```

### Install dependencies

```bash
npm install
```

### Run development server

```bash
npm start
```

### Run docs

```bash
npm run docs
```
